# Photos folder

This folder is empty on purpose — placeholder or stock photos would look generic
and would undercut the "we've actually done this work" credibility that matters
most for NexBridge. Real photography from your own projects will always beat
stock imagery here.

When you have real photos, this is where they go. Suggested files and sizes:

- `founder.jpg` — your headshot for the About > Founder page. Square, at least
  800x800px.
- `factory-01.jpg`, `factory-02.jpg`, etc. — shop floor / production line shots
  for Industries and Knowledge articles. Landscape, at least 1600px wide.
- `case-study-[name]-01.jpg` — photos tied to specific case studies, named by
  project so they're easy to find later.
- `team.jpg` — group or office photo for About.

Keep filenames lowercase with hyphens (no spaces) so they work reliably as URLs
once the site is live.
