# Banner folder

The homepage hero currently uses a CSS/SVG animated network background — no
image file needed, loads instantly, and looks sharp at any screen size.

If you'd rather use a real photo here later (e.g. a shot from your own
factory floor or a project site), drop it in as `hero-banner.jpg` (at least
1800px wide) and let me know — I'll swap the CSS background for the photo
with a dark overlay so the white headline text still reads clearly on top.
